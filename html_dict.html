Dicionário de Elementos HTML

Este é um guia das tags HTML utilizadas na construção deste site.

<!DOCTYPE html>

Declaração que define o tipo de documento como HTML5. Não é uma tag HTML, mas uma instrução para o navegador sobre qual versão do HTML está sendo usada.

<!DOCTYPE html>

<html>

Elemento raiz que contém todo o conteúdo da página HTML.

<html lang="pt-BR">...</html>

<head>

Contém metadados sobre o documento, como título, links para CSS, meta tags, etc.

<head>...</head>

<meta>

Define metadados como codificação de caracteres, descrição da página, palavras-chave, etc.

<meta charset="UTF-8">

<title>

Define o título da página que aparece na aba do navegador.

<title>Davi Turnes Vieira | Front-end Developer</title>

<link>

Conecta o documento a recursos externos, como folhas de estilo CSS.

<link rel="stylesheet" href="style.css">

<body>

Contém todo o conteúdo visível da página.

<body>...</body>

<header>

Define o cabeçalho da página, geralmente contendo logo, navegação e controles.

<header>...</header>

<nav>

Define uma seção de navegação com links para outras páginas ou seções.

<nav>...</nav>

<menu>

Representa um grupo de comandos ou opções, usado para criar menus de navegação.

<menu>...</menu>

<li>

Define um item de lista, usado dentro de elementos ul, ol ou menu.

<li>Item da lista</li>

<a>

Cria um link para outra página, arquivo ou local na mesma página.

<a href="index.html">Link para a página inicial</a>

<aside>

Define conteúdo relacionado mas separado do conteúdo principal, como barras laterais.

<aside>...</aside>

<main>

Especifica o conteúdo principal do documento, excluindo cabeçalhos, rodapés e barras laterais.

<main>...</main>

<section>

Define uma seção temática no documento, como capítulos, abas ou partes de uma página.

<section id="section-home">...</section>

<h1> a <h6>

Definem títulos de diferentes níveis, onde h1 é o mais importante e h6 o menos importante.

<h1>Título principal</h1>

<p>

Define um parágrafo de texto.

<p>Este é um parágrafo de texto.</p>

<div>

Define uma divisão ou seção genérica em um documento, usado para agrupar elementos.

<div class="container">...</div>

<span>

Define uma seção inline em um documento, usado para aplicar estilos a partes específicas de texto.

<span class="highlight">texto destacado</span>

<img>

Insere uma imagem no documento.

<img src="imagem.jpg" alt="Descrição da imagem">

<button>

Cria um botão clicável.

<button type="button">Clique aqui</button>

<i>

Originalmente usado para texto em itálico, agora comumente usado para ícones (como Font Awesome).

<i class="fas fa-home"></i>

<footer>

Define o rodapé de uma página ou seção.

<footer>...</footer>

<script>

Usado para incorporar ou referenciar código JavaScript.

<script src="script.js"></script>
