Dicionário de Elementos HTML

Este é um guia das tags HTML utilizadas na construção deste site.

<!DOCTYPE html>

Declaração que define o tipo de documento como HTML5. Não é uma tag HTML, mas uma instrução para o navegador sobre qual versão do HTML está sendo usada.

<!DOCTYPE html>

<html>

Elemento raiz de um documento HTML. Todos os outros elementos devem ser descendentes deste elemento.

<html lang="pt-BR">...</html>

<head>

Contém metadados e informações sobre o documento HTML, como título, links para estilos, scripts e metadados.

<head>...</head>

<body>

Contém o conteúdo visível do documento HTML, incluindo texto, imagens, links e outros elementos.

<body>...</body>

<header>

Representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação. Geralmente contém o cabeçalho da página, logo e menu de navegação.

<header>...</header>

<nav>

Define uma seção de navegação. Usado para criar menus de navegação com links para outras páginas ou seções do site.

<nav>...</nav>

<main>

Especifica o conteúdo principal do documento. Deve ser único no documento e não deve conter conteúdos repetidos como cabeçalhos, rodapés ou barras laterais.

<main>...</main>

<section>

Define uma seção em um documento, como capítulos, cabeçalhos ou quaisquer áreas independentes de conteúdo.

<section>...</section>

<aside>

Define um conteúdo relacionado mas separado do conteúdo principal. É geralmente representado como uma barra lateral.

<aside>...</aside>

<footer>

Define um rodapé para um documento ou seção. Geralmente contém informações de autoria, direitos autorais, links de contato e navegação secundária.

<footer>...</footer>

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>

Tags de cabeçalho que definem títulos e subtítulos. H1 é o mais importante, e H6 o menos importante.

<h1>Título Principal</h1>
<h2>Subtítulo</h2>

<p>

Define um parágrafo de texto.

<p>Este é um parágrafo.</p>

<a>

Define um hiperlink, usado para navegar entre páginas.

<a href="https://www.exemplo.com">Link para exemplo</a>

<img>

Insere uma imagem no documento.

<img src="imagem.jpg" alt="Descrição da imagem">

<ul>

Define uma lista não ordenada (com marcadores).

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>

<li>

Define um item de lista, usado dentro de elementos <ul> ou <ol>.

<li>Item da lista</li>

<div>

Define uma divisão ou seção genérica em um documento. Usado como contêiner para outros elementos HTML.

<div>...</div>

<span>

Define uma seção em linha em um documento, usado para agrupar elementos em nível de texto.

<span>texto em destaque</span>

<form>

Define um formulário HTML para entrada do usuário.

<form>...</form>

<input>

Define um campo de entrada onde o usuário pode inserir dados.

<input type="text" name="nome" placeholder="Digite seu nome">

<textarea>

Define um controle de entrada de texto de várias linhas.

<textarea name="mensagem" rows="4" cols="50">...</textarea>

<button>

Define um botão clicável.

<button type="submit">Enviar</button>

<script>

Define um script do lado do cliente, geralmente JavaScript.

<script src="script.js"></script>

<link>

Define a relação entre um documento e um recurso externo, mais comumente usado para vincular folhas de estilo CSS.

<link rel="stylesheet" href="style.css">

<meta>

Define metadados sobre um documento HTML, como descrições, palavras-chave para mecanismos de busca, autor do documento e configuração de viewport.

<meta name="description" content="Descrição da página">

<address>

Define informações de contato do autor/proprietário de um documento ou artigo.

<address>
